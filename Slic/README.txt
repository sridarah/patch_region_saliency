This directory contains the public implementation of "Slic superpixels".

Please cite the following publication if you use this C++ code or the compiled Mex files

R. Achanta, A. Shaji, K. Smith, A. Lucchi, P. Fua, and S. Susstrunk, 
“SLIC superpixels compared to state-of-the-art superpixel methods,”
IEEE TPAMI, vol. 34, no. 11, pp. 2274–2282, 2012.

or

R. Achanta, A. Shaji, K. Smith, A. Lucchi, P. Fua, and S. Susstrunk, 
"Slic superpixels," Technical Report, EPFL, 2010.
